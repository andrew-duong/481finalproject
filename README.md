To use the system as a staff member, log in using one of the staff logins. To use the system as a parent, log in using one of the parent logins.
Scenarios 1-6 listed in design portfolio 1 were implemented in the system.

User Scenarios
Scenario 1 – Staff Adds a Log

Tap Log In and sign into the system using any staff account.

Press Log on the bottom navigation bar. You will be redirected to the Add Logs page.

Open the Select Children dropdown and check the children to include in the log. A Behavioural Notes section will appear.

From the Activity Name dropdown, select Arts & Crafts.

Adjust the Start Time and End Time as needed.

Enter a note in the General Notes field.

To add behavioural notes for a specific child:

Tap the + icon.

Select a child and enter the note. The note will save automatically.

Tap Submit to finalize the log.

You will be redirected to the Logged Activities page.

The newly added log should appear in the list.

Scenario 2 – Parent Views Daily Activity

Tap Log In and sign into the system using any parent account.

Press Activity on the bottom navigation bar. You will be redirected to the My Children page.

Select a child from the list to open their Daily Activity page.

Tap the dropdown arrow on an activity card to expand it.

The general notes and any behavioural notes will be displayed.

Scenario 3 – Parent Views Events

Tap Log In and sign into the system using any parent account.

On the Home page, a Calendar View and an Events This Month section will appear.

Press Events on the bottom navigation bar to open the Upcoming Events page.

If multiple children are associated with the account, select a child using the child selector.

A list of events for that child will be displayed.

Tap Details on any event to open the Event Details page.

The page will display:

Location

Date & Time

Notes

Description

Fee (if applicable)

At the bottom, a button for Form and/or Payment will appear:

Paid → disabled

Filled → disabled

Tap the form button to be redirected to the Form Page.

Scenario 4 – Staff Creates an Event

Tap Log In and sign into the system using any staff account.

Press Events on the bottom navigation bar to open the Upcoming Events page.

Tap Add Event to open the Add Event page.

Enter all required fields:

Event Title

Location

Date

Selected Children

Start & End Time

Enter optional Description and Notes.

Attach a form if needed using the Upload section.

Tap Submit to create the event.

You will be redirected back to the Events page, and the newly created event will appear in the list.

Scenario 5 – Parent Fills Out a Form

Tap Log In and sign into the system using any parent account.

Press Form on the bottom navigation bar to open the Forms page.

If multiple children are linked to the account, select a child using the child selector.

A list of forms for that child will appear. Form statuses:

Outstanding → active Fill button

Pending → disabled Filled button

Completed → disabled Filled button

Tap Fill on any Outstanding form.

The Form Details page will display a Download Form option.

Fill out all required fields:

Parent Name

Emergency Contact

E-Signature

Consent

Add optional notes if needed.

Tap Submit Form.

The form will return to the list with a Pending status.

Scenario 6 – Parent Pays a Fee

Tap Log In and sign into the system using any parent account.

Press Pay on the bottom navigation bar to open the Fees page.

If multiple children are associated with the account, select a child using the child selector.

A list of fees for that child will appear. Fee statuses:

Outstanding → active Pay Now button

Pending → disabled Paid button

Completed → disabled Paid button

Tap Pay Now on any Outstanding fee.

Fill in the payment form:

Cardholder Name

Card Number

Expiry Date

CVV

Tap Complete Payment.

You will be redirected to the Payment Confirmation page.
